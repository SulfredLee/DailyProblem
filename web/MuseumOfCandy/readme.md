# Museum Of Candy Project

## What have I learnt

### CSS

#### How to use CSS to style?
We can select the target by id and class name
```css
#mainNavbar .nav-link {
    color: white;
}
```

By id and type name
```css
#headingGroup h1 {
    font-weight: 100;
}
```

By class and type name
```css
.blurb h2 {
    font-size: 2rem;
}
```

#### Use media query to control the style based on the document width
```css
@media (max-width: 1200px) {
    #headingGroup h1 {
        font-weight: 100;
        font-size: 3rem;
    }
    .blurb h2 {
        font-size: 2rem;
    }
}
```
